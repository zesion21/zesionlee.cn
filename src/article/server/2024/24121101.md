# 为 APT 包管理器设置 XX

`sudo vim /etc/apt/apt.conf.d/80proxy`

```bash
Acquire::http::proxy "http://10.10.1.20:8080/";
Acquire::https::proxy "http://10.10.1.20:8080/";
# Acquire::ftp::proxy "ftp://10.10.1.20:8080/";
```
