# ubuntu 固定 IPv4 上网

修改配置文件 `sudo vim /etc/netplan/50-cloud-init.yaml`

```bash
network:
    ethernets:
        ens33:
            dhcp4: false  # 关闭DHCP
            addresses:
              - 192.168.83.150/24  # IP地址
            gateway4: 192.168.83.1 # 网关
            nameservers:
              addresses: [223.5.5.5,8.8.8.8] # DNS
    version: 2


```

应用修改 `sudo netplan apply`
