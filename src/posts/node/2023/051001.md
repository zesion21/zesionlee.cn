### superagent 发送 Post 请求

1. json 类型

```js
superagent
  .post("/api/pet")
  .send({ name: "Manny", species: "cat" }) // sends a JSON post body
  .set("X-API-Key", "foobar")
  .set("accept", "json")
  .end((err, res) => {
    // Calling the end function will send the request
  });
```

2. form 类型

```js
superagent
  .post("/api/pet")
  .send({ name: "Manny", species: "cat" })
  .type("form")
  .end((err, res) => {
    // Calling the end function will send the request
  });
```
