在原来 JDK8 下，想要在 geoserver 添加可用的 font 是将字体文件添加放到

```
%JAVA_HOME%/jre/lib/fonts
```

而在 JDK11 时，Oracle JDK 不再提供任何字体，完全依赖于操作系统上安装的字体

所以只需要将字体安装到操作系统便能在 geoserver 中使用
